# Веб-приложение для лабораторной работы №2

## Описание
Веб-приложение на Flask.

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone https://github.com/yamazakk1/web_labs
cd web_labs 
```

2. Создать виртуальное окружение:

```bash
python -m venv ve
```

3. Активировать окружение:

Windows:

```bash
.\ve\Scripts\activate
```

Linux/Mac:

```bash
source ve/bin/activate
```

4. Установить зависимости:

```bash
pip install -r requirements.txt
```
5. Запустить приложение:

```bash
python app.py
```
6. Открыть в браузере:
```
http://127.0.0.1:5000
```
